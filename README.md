# 🐍 Snake, 💧 Water, 🔫 Gun Game (Python)

A simple, interactive command-line game similar to Rock, Paper, Scissors — built using Python.

---

## 🚀 Features

- Dictionary-based game choice mapping
- Randomized computer moves using the `random` module
- Instant win/loss evaluation based on game rules

---

## 🎮 Rules

|  You choose   |   Beats    |       Because      |
|---------------|----------  |--------------------|
| 🐍 Snake     | 💧 Water   | Snake drinks water |
| 💧 Water     | 🔫 Gun     | Water drowns/rusts the gun |
| 🔫 Gun       | 🐍 Snake   | Gun shoots the snake |

### Input Choices:
- Enter `s` → Snake 🐍
- Enter `w` → Water 💧
- Enter `g` → Gun 🔫

---

## 🛠️ How to Run Locally

1. **Clone the repository:**
```bash
git clone https://github.com/Khushi6307/snake-water-gun.git
```

2. **Navigate to the directory:**
```bash
cd snake-water-gun
```

3. **Run the script:**
```bash
python main.py
```
